# Search Filter (JavaScript)

A real-time search/filter UI built with vanilla JavaScript. Type in the search box to instantly filter a list of names.

## How It Works

- A list of names is displayed on the page
- As you type in the search input, names are filtered in real time
- Uses the `includes()` method for case-insensitive substring matching
- Non-matching names are hidden via CSS `display: none`

## Tech Stack

- HTML, CSS, JavaScript

## How to Run

Open `index.html` in any web browser.
